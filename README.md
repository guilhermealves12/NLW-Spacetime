## 🖥️ Projeto

Esse é um projeto Web responsivel de uma capsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante o NLW da Rockeatseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
